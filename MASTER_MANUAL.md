### How to Build a New Feature

1. **Request**: Tell me what you want to build.
2. **Backend**: I add logic to `mobile/convex/`.
3. **Frontend**: I update `mobile/App.js` or `web/src/app/page.tsx`.
4. **Visual Check**: I use `browser_subagent` to verify the UI.
5. **Vibe Check**: You see the update instantly on your phone via **Expo Go**.

### How to Start the Lab

Open terminals in different directories:

- **Cloud Backend**: `npx convex dev` (in `mobile/`).
- **Mobile Frontend**: `npx expo start --clear` (in `mobile/`).
- **Web Frontend**: `npm run dev` (in `web/`).
